  2. Second Project Idea At this point we had done some brainstorming and we had come up with the idea to have a pysical robot that takes a picture of our users face and draws it.
  3. Third Project IdeaTakes a picture of the User, the picture will go into p5.js. The image would be sent back up to the robot to paint the distorted picture.
  4. Fourth Project Idea We left the idea of the robot to work to focus on an Idea of applying filters in p5. We also thought about the filtered image to be turned into a stencil that the user could fill out.
  5. Fifth Project Idea We left behind the idea of the robot and descided to switch to python instead of p5. We did this for the speed of the program as well as the recourses already provided in python.
  6. Sixth Project Idea We found a beautification algorithm on GitHub with good documentation and easy installation with python.
